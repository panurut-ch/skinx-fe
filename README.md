## Library
- React
- Next.js
- Material UI
- Mui-datatables
- Axios

## How to run local
```bash
npm install
```
```bash
npm run dev
```
Web will live on
```bash
http://localhost:3001/
```

## How to run with Docker
```bash
docker-compose build --no-cache
```
```bash
docker-compose up -d
```
Web will live on
```bash
http://localhost:3001/
```
